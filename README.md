# Razzle ✨

A zero-config, razzle-dazzle build system for universal javascript applications.

## Features

 - 🌎 **Universal Hotness** Both the client and server use hot module replacement.
 - ⚛️ **Babel / Webpack Goodness**
 - 🚫 **Zero-Config*ness*?**


## Quick Overview

```bash
yarn add --global razzle

razzle new myApp
cd myApp
yarn start
```
